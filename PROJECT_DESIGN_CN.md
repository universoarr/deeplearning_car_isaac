# Isaac Lab 小车智能行驶项目设计

## 1. 项目目标

在 Isaac Lab 中训练一台带有 IMU 六轴传感器和前视摄像头的小车，使其在复杂地形中具备以下能力：

- 在崎岖、碎石、轻微坡地等非平整路面上保持稳定直线行驶
- 识别前方障碍并自主绕行
- 在存在扰动、打滑、颠簸的情况下维持较好的姿态与方向控制
- 具备一定泛化能力，从训练地形迁移到未见过的复杂场景

## 2.任务：感知驱动避障导航

目标：

- 在前方有石块、路障、坑洼边缘时绕行
- 绕开障碍后继续回到主方向

输入：

- 前视相机图像
- 6轴IMU
- 航向误差

输出：

- 动作：`[leftPWM, rightPWM]`

## 5. 奖励函数设计

正奖励：

- 前向速度接近期望值
- 成功绕开障碍
- 航向误差小
- 横向偏移小
- 恢复到主路径

惩罚：

- 过大转向
- 频繁动作抖动
- 横摆角速度过大
- 长时间停滞
- 撞击障碍
- 距障碍过近
- 原地打转
- 无效绕远

可以写成加权和：

```text
reward =
  + w1 * forward_velocity
  - w2 * heading_error
  - w3 * lateral_offset
  - w4 * yaw_rate
  - w5 * action_smoothness
  - w6 * roll_pitch_penalty
```

## 8. 终止条件

- 车辆翻倒
- 碰撞强度超过阈值
- 偏离任务区域
- 长时间没有前进
- 超过最大 episode 长度
